# My-first-commit-
Test space invaders 
A browser-based Space Invaders clone built with HTML5 Canvas and the Web Audio API.
  Defend Earth against four waves of alien attackers — Squids, Crabs, Saucers, and Bugs —
  each with unique pixel-art animations. Destroy entire waves to advance levels, shoot down
  a powerful Boss alien every third level, and take cover behind destructible barriers.
  Score multiplies with harder aliens and boss kills. Your high score is saved automatically
  between sessions. Use ← → to move, Space to shoot, and Enter to start.
